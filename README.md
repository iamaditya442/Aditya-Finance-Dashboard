# 💰 Finance Dashboard UI

## 📌 Project Overview

This project is a Finance Dashboard UI built as part of a frontend assignment. It focuses on creating a clean, simple, and interactive interface for users to understand their financial data.

The project is frontend-only and uses mock data to simulate real-world financial tracking.

## 🎯 Features

### 📊 Dashboard
- Total Balance, Income, Expenses cards  
- Time-based chart (balance trend)  
- Category-based chart (spending breakdown)  

### 📄 Transactions
- View transaction details:
  - Date  
  - Amount  
  - Category  
  - Type (Income/Expense)  
- Search, filter, and sort functionality  

### 🔐 Role-Based UI
- Viewer → View only  
- Admin → Add/Edit transactions  
- Role switch using dropdown  

### 📈 Insights
- Highest spending category  
- Monthly comparison  
- Basic financial insights  

---

## ⚙️ State Management

Handles:
- Transactions data  
- Filters and search  
- User role

## 🎨 UI & UX

- Clean and minimal design  
- Fully responsive  
- Smooth user interaction
