# 💰 Finance Dashboard UI

## 📌 Project Overview

This project is a Finance Dashboard UI built as part of a frontend assignment. The goal was to create a clean, simple, and interactive interface that helps users understand their financial data easily.

The focus is on frontend development, including UI design, component structure, and state management. No backend is used — all data is static or mock data.

---

## 🎯 What This Project Does

The dashboard allows users to:

- View overall financial summary  
- Explore transactions  
- Understand spending patterns  
- Switch between different user roles (Viewer/Admin)  

---

## ✨ Features

### 📊 Dashboard Section
- Summary cards:
  - Total Balance  
  - Total Income  
  - Total Expenses  
- Includes:
  - Time-based chart (balance trend)  
  - Category-based chart (spending breakdown)  

---

### 📄 Transactions Section
- Displays transactions with:
  - Date  
  - Amount  
  - Category  
  - Type (Income / Expense)  
- Functionalities:
  - Search  
  - Filter  
  - Sort  

---

### 🔐 Role-Based UI (Frontend Only)
- Viewer → Can only view data  
- Admin → Can add/edit transactions  
- Role switching via dropdown or toggle  

---

### 📈 Insights Section
- Highest spending category  
- Monthly comparison  
- Basic financial observations  

---

### ⚙️ State Management
Handles:
- Transactions data  
- Filters and search  
- Selected role  

Implemented using a clean frontend state approach.

---

## 🎨 UI & UX

- Clean and minimal design  
- Easy to navigate  
- Responsive across devices  
- Handles empty states properly  

---

## 🛠️ Tech Stack

- Frontend: React / JavaScript  
- Styling: CSS / Tailwind / MUI  
- State Management: useState / Context API
